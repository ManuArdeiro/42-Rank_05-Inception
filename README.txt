
PASOS PARA ARRANCAR

1. Coloca este proyecto en ~/inception
2. Ejecuta:

make build
make up

3. Accede a https://jolopez-.42.fr para completar la instalación de Wordpress

IMPORTANTE:
- Si ves un 403 Forbidden es que Wordpress no se ha copiado o no tiene permisos. 
- Este pack ya incluye permisos correctos.
- No necesitas tocar nada más, está todo listo.
